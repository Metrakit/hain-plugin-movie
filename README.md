A [Hain](https://github.com/appetizermonster/hain) plugin for find a movie !

Search fastly a movie from the [IMDb](http://imdb.com) website in Hain. 

## Install

-  Type this command in the Hain input :
```
/hpm install movie
```

## Usage

```
/movie Avengers
```
